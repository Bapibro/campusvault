# CampusVault

A modern student marketplace frontend built with Next.js, React, Tailwind CSS, and TypeScript.

## Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000` in your browser.

## Pages included

- `/` – Homepage
- `/marketplace` – Marketplace catalog
- `/login` – Login page
- `/signup` – Signup page
- `/dashboard` – User dashboard
- `/admin` – Admin dashboard
- `/payment` – UPI payment flow guide

## Notes

- This is a frontend-first project with mock data and reusable components.
- The checkout flow uses a simple UPI payment guide for manual access control.
- Use this starter to build next features like authentication, backend APIs, and admin uploads.
